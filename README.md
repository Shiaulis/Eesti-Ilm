## About the App

Eesti Ilm is a simple weather app,that shows infromation from [Estonian Weather Service](http://www.ilmateenistus.ee/?lang=en)

## App Releases

### v.1.2
The very first AppStore release. Contains 4-days forecast screen + 2 widgets
